# 说明

* 快速搭建项目
* vue-vite
* vue-vite-ts
* vue2
* vue3
* vue3-ts

## 使用

* 全局安装：`npm i -g auc-cli`
* 可选模板
  * vue-vite
  * vue-vite-ts
  * vue2
  * vue3
  * vue3-ts
* 命令：`ac [project-name]` 或 `ac [project-name] -t=[template-name]`
  * 例如：`ac my-project` 或 `ac my-project -t=vue3`（选择vue3模板）
* 创建好之后：`cd [project-name]`
* `npm install` 或 `yarn`
* `npm run dev` 或 `yarn dev`
